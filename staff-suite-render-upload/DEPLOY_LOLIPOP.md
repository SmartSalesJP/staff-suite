# ロリポップ公開メモ

このアプリは `server.js` を常時起動する Node.js アプリです。
HTML/CSS/JS だけをアップロードする静的サイトではないため、ロリポップ側で Node.js を起動し続けられる環境が必要です。

## アップロード前に必要な情報

- ロリポップのプラン
- 公開するドメインまたはサブドメイン
- SFTP/SSH ホスト名
- SSH/SFTP ユーザー名
- SSH/SFTP パスワード、または秘密鍵
- 公開先ディレクトリ
- Node.js を起動できるか
- 起動コマンドを常駐化できるか

## 起動コマンド

```bash
npm start
```

または

```bash
PORT=3000 node server.js
```

## アップロードする主なファイル

- `server.js`
- `package.json`
- `public/`
- `data/`

## 注意

現在のデータ保存先は `data/db.json` です。複数人で本番利用する場合は、将来的に MySQL などへ移行する方が安全です。

公開前に必ず管理者パスワードを変更してください。
